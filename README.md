# Bigni-Oubo Football Académie

Small website for the academy — mission, gallery, partners, and links to the social pages. Nothing fancy under the hood: it’s plain HTML and CSS (with Bootstrap for the grid and carousel), so you can open it anywhere without installing anything.

## Run it locally

Open `index.html` in your browser (double-click the file or drag it into a tab). That’s all you need for this project.

## What’s in the folder

- `index.html` — all the sections and content  
- `CSS/styles.css` — layout and look  
- `Images/` — logos, photos for the gallery 

If an image path breaks after you move files, check that the `Images` folder name matches what’s in the HTML (capital **I** matters on some hosts).
